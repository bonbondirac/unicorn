　　广东宏远俱乐部26日向中国篮协递交报告，宣布本队球员易建联将放弃本年度 N BA选秀的资格。中国篮协已同意俱乐部的这一决定。
　　中国篮协26日公布了这一消息。篮管中心副主任胡加时说：“经过广东宏远俱乐部全体董事会成员集体表决，易建联放弃今年的 N BA选秀，中国篮协同意俱乐部的决定。”
　　之前一直在传易建联将参加 N BA选秀，并且易建联的经纪人也出来预测易建联选秀的情况，最终证明都是炒作。胡加时说：“易建联放弃选秀的主要原因是认为他进入 N BA的时机和条件都不成熟，留在国内对小易更有好处。宏远俱乐部为此事也咨询了很多专家，认为易建联去 N BA打球的机会不大，即便去了上场时间也不会太多，反倒是 C BA和国家队更能给他机会和空间。易建联本人也同意俱乐部的决定。”
　　谈到中国球员参加 N BA选秀，胡加时说中国篮协还没有一个专门针对选秀的规定，只有一个《运动员涉外转会》的规定。篮协对中国球员选秀的态度是“大门开放，态度积极”，“本着三个有利原则，即有利于国家队备战2008奥运会，有利于俱乐部建设和发展，有利于球员本身的成长，我们允许运动员参加选秀”。
　　另外，最近还盛传奥神俱乐部的孙悦也要参加 N BA选秀。对此，胡加时说，至今他们还没有收到奥神俱乐部的正式通知，因此无法对孙悦的事下结论，“但无论是什么决定，只要运动员本人同意，俱乐部同意，我们都尊重他们的意见”。
　　N BA选秀22岁以下球员报名到28日截止，中国国家男篮的集训大名单也将于本月底公布。
