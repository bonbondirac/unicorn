　　【eNet硅谷动力消息】当地时间本周三，微软公司宣布已为SQL Server 2005发布了第一个服务包，新的服务包修正了许多缺陷，为企业数据库软件最新升级铺平了道路。
　　微软公司表示，在发布SQL Server 2005五个月后这个数据库将进行新的升级，这意味着微软公司的工程师正在与数据库领域的领先厂商甲骨文公司以及IBM公司的数据库产品进行着激烈竞争。
　　英国软件开发商自由撰稿人David Cartwright 表示，许多企业并没有急忙部署这个重大产品升级的 第一个版本，它们更喜欢让少数用户对这一产品进行试验。他对这个数据库进行了评估，在通常情况下，一个最初的服务包在发布后将引发企业开始进行升级。
　　据Cartwright表示，迄今为止，SQL Server 2005 已经被证实是一款可靠的产品，他说：“至今我听说没有出现令人不愉快的事情。”
　　微软公司表示，这个服务包彻底修正了大约40个缺陷，并整合了几个新的功能，其中包括生产就绪（production-ready ）数据库镜像功能，它强化了第一个版本的价值。数据库镜像就是在一个案例中数据库就绪的拷贝，已经有20个微软公司的客户使用了镜像功能。
　　微软公司指出，新的服务包在SQL Server 2005的报告服务中增加了两个组件，能够支持企业利用SAP公司的商务智能软件进行报告，这个功能允许对 SAP企业数据库内部的报告进行创作和管理。
　　数据库的免费版本SQL Server 2005 Express Edition 同样强化了绘图管理工具，管理员不用访问特殊的开发工具，在DOS窗口利用一条指令就可以移动到先前的数据库版本。
　　 微软公司表示，新的服务包推出后，预期SQL Server 2005将能够帮助微软公司更加有效的和甲骨文以及IBM公司在数据库产品领域展开竞争。
