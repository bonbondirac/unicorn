　　4月10日，可编程逻辑解决方案提供商赛灵思公司宣布推出用于嵌入式处理设计的赛灵思平台Studio 工具套件 8.1i （Platform Studio 8.1i） 版本。这一最新版本整合了一种新的图形用户界面，通过使通用的、重复性任务变得轻松和直观，从而简化了基于平台的设计。Platform Studio 8.1i 还提供了丰富的适用性功能，使用 PowerPC&#8482; 硬处理器核和 MicroBlaze&#8482; 软处理器核进一步简化、抽象和加快了嵌入式系统的开发。
　　“嵌入式工程师希望能够在开发过程中使用所有设计选项和特性。但更为重要的是，他们需要把系统尽快建立起来并投入运行，”赛灵思公司嵌入式处理与 IP 部门副总裁 Mark Aaldering 说，“我们不断地把更多的智能和自动化功能嵌入到我们的平台Studio 工具中，从而使开发者能够节省时间，并将其资源集中于更快地向市场推出最高质量的差异化系统。”
　　直观的工具套件
　　8.1i 版本所拥有的改进的基于菜单的图形用户界面和适用性功能，涵盖了工具套件的几乎所有方面，使嵌入式开发者能够更容易更快速地构建符合其要求的系统。新的系统框图查看器使工程师能够以可视化的方式观察整个系统层次，然后使用设计报告生成器自动创建详细的设计文档。新的系统组装面板提供了采用颜色编码的图形化连接性视图，以辅助总线连接编辑，该视图可与“点击即连”特性一起使用，以防止出现不兼容的连接。简化的过滤功能也使开发者能够快速创建定制设计视图。其它增强功能还包括新的直观的知识产权 (IP) 核配置对话框以及IP 核搜索功能。
　　快速启动开发
　　为了快速启动设计进程，平台Studio 8.1i 通过其 Base System Builder 向导为赛灵思开发板（包括对 MicroBlaze 处理器的浮点支持）以及来自无数嵌入式合作伙伴的参考与评估板提供自动化板支持。新的仿真与调试特性支持使用硬件调试器和逻辑分析仪进行执行跟踪，同时也支持来自硬件逻辑仿真器的处理器寄存器视图。此外，8.1i 版本还提供了对 C++ 编译器/链接器和脚本以及新网络安装功能的增强支持。
　　平台、价格和供货情况
　　平台Studio 8.1i 支持使用赛灵思 Virtex&#8482;-4、Virtex-II Pro、Spartan&#8482;-3 和 Spartan-3E 平台 FPGA 实现 PowerPC 和 MicroBlaze 处理器设计。可以直接用于以下计算平台：Windowsreg; 2000 SP2、SP3、SP4，和 XP SP1、SP2，Solarisreg; 2.8/5.8 和 2.9/5.9，以及 Linuxreg; Red Hatreg; Enterprise 3.0。
　　平台Studio 套件在 8.1i 版赛灵思嵌入式开发套件 (EDK) 中方便地捆绑了一个处理器 IP 库、软件驱动程序、文档、参考设计和 MicroBlaze 软处理器 IP 核，售价 495 美元。还包括赛灵思集成软件环境 (ISE&#8482;) 8.1i 逻辑工具、ChipScope&Ocirc; Pro 硬件调试器及大量第三方工具和技术的评估版。
