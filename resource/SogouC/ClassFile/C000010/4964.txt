　　――升腾资讯作为国内唯一获微软金牌认证的终端厂商与会
　　本报讯 3月28日，微软服务器操作系统――windows server 2003 R2新品发布会在北京长城饭店隆重举行，来自业界的企业精英、著名专家学者等300多人一起出席了本次盛会。作为此次大会唯一受邀请的终端厂商，升腾资讯在会上充分展出了其国内领先的升腾瘦客户机－服务器整体解决方案，受到了与会嘉宾的极大关注。
　　长期以来，升腾资讯与微软建立了密切的战略合作伙伴关系，特别是在Windows终端技术方面，双方有着良好而成功的合作基础。从98年国内第一台Windows终端DD升腾2000成功推出至今，双方技术人员始终保持着深层次多方位的交流，每一代升腾windows终端产品的推出都蕴涵了微软同期最新的技术成果，今年年初，升腾资讯更是凭借出色的高级基础架构解决方案能力和网络基础架构解决方案能力，成为国内唯一一家获得微软金牌认证合作伙伴的终端厂商。
　　前期技术调试表明：升腾Windows终端完全符合Windows Server 2003 R2终端服务标准的产品，其优越的性能充分发挥了Windows Server 2003 R2的功能。随着双方合作的进一步深入，升腾资讯将从微软公司得到超越其他终端厂商的资源和技术支持，并结合自身的自主创新能力，为客户带来更先进的终端产品、解决方案和更卓越应用体验。
　　会后，升腾资讯总经理郑宏说：升腾资讯拥有强大的技术实力和长期的科研沉淀，企业自我研发能力强，随着微软Windows Server 2003 R2的成功发布，升腾资讯将与微软强强联手，继续推进Windows瘦客户机的计算模式在中国的推广与应用，为中国信息化创新建设做出持续性贡献。
